Keep Your Mac
Awake

The elegant menu bar app that prevents your Mac from sleeping. Simple, native, and lightweight.

Download for macOS View on GitHub
Requires macOS 11.0+

Everything you need, nothing you don't

Built for simplicity. Designed to stay out of your way.

One-Click Toggle

Click the menu bar icon to instantly toggle. No complicated menus or settings required.

Duration Timer

Set for 15 minutes, 1 hour, 3 hours, custom duration, or keep it running indefinitely.

Display Control

Keep the system awake or prevent the display from sleeping too. Complete control is yours.

Global Shortcuts

Configure custom keyboard shortcuts to toggle CaffeineMate from anywhere on your Mac.

Launch at Login

Start automatically when you log in. Always ready when you need it, invisible when you don't.

Native & Lightweight

Built with Swift and SwiftUI. No Electron, no bloat. Just a tiny, efficient native app.

Install in seconds

Get started with a single command using Homebrew

Terminal
$ brew install eddmann/tap/caffeinemate

Or download directly from GitHub Releases
